Arkan Quest

Jul 2019

An Arkanoid-like game that I made in Construct 3 and art in Photoshop just for learning purposes. I met the classic Arkanoid game as a child and I always liked the mechanics of it. It is a challenging and relatively simple game to play. While on vacation I was learning Construct 3 and decided to make a game for mobile devices with him. I needed a simple game to implement, that would work on any mobile device, I also wanted to venture into the artistic world creating all the assets and, in addition, I wanted to have fun creating and testing this game. Making an Arkanoid clone was perfect for me to learn how to develop on Construct 3, create assets in Photoshop and test on my own phone (which was anything but potent). I added a personal touch to the game with black hole mechanics and the game was really fun to play.


What I Learned

Developing this project I learned how to make games using Construct 3, assets with Photoshop, balancing and optimizing mechanics for games on mobile devices, progressing levels and increasing difficulty in game design.
